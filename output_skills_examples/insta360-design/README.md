# Insta360 - Design Style Guide

Extracted from: https://www.insta360.com/cn/

## Overview

This design style guide captures the visual design system of the Insta360 official website. Insta360 (影石) features a dark, cinematic aesthetic with full-bleed product imagery, pill-shaped CTAs, and premium consumer electronics branding.

## Key Design Features

- **Dark premium palette**: Near-black `#262629` primary action color on pure black backgrounds
- **Pill CTA buttons**: 50px border-radius as the signature UI element across all actions
- **Cinematic imagery**: Full-bleed hero images with 15-step gradient overlays
- **Poppins + Roboto fonts**: Variable font weights for title vs content typography
- **Subtle hover animations**: 0.2-0.3s transitions with image zoom (1.05-1.1x scale)
- **Gold gradient chat button**: Unique customer service CTA
