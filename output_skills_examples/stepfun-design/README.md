# 阶跃星辰 StepFun 设计风格提取

## 项目说明

本项目是从 [阶跃星辰 StepFun](https://www.stepfun.com/chats/new) 对话页面提取的设计风格指南，采用网页设计风格提取器工具自动提取并人工验证。

## 提取信息

- **源网站**：https://www.stepfun.com/chats/new
- **提取时间**：2026-04-16
- **行业**：科技 / AI 对话
- **框架**：Next.js + Tailwind CSS + shadcn/ui
- **主字体**：Gilroy

## 目录结构

```
stepfun-design/
  ├── SKILL.md                # 设计风格指南（核心输出）
  ├── README.md               # 本文件
  ├── result_validation.html  # 验证页面
  └── assets/                 # 提取数据
      ├── page-info.json      # 完整页面信息数据
      ├── extraction-summary  # 提取摘要
      └── page-screenshot.png # 页面截图
```
