# Bambu Lab 拓竹科技 设计风格指南

## 项目说明

本项目是从 [Bambu Lab 官网](https://bambulab.com/zh) 提取的设计风格指南，适用于3D打印/制造科技行业网站页面设计。

## 行业定位

**科技行业 - 3D打印/制造科技**

Bambu Lab（拓竹科技）是一家专注于3D打印技术的科技公司，其官网设计风格集中体现了高端科技品牌的视觉特征。

## 设计风格概述

- **核心色彩**：品牌绿 (#00AE42) + 深黑 (#1A1A1A) + 纯白 (#FFFFFF)
- **主字体**：Manrope
- **设计风格**：现代、简约、高端科技感
- **布局特征**：沉浸式大图、纵向滚动产品展示、3列产品网格

## 文件结构
